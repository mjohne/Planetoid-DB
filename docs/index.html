About Planetoid-DB

Planetoid-DB reads the large ASCII database of the Minor Planet Center Orbit Table (MPCORB.DAT), which contains orbital data of hundreds of thousands of small celestial bodies (including orbital elements, absolute brightness, latest observation data, etc.) and displays it in a graphical interface.

This file is the official MPC database for minor planets and comets and contains, among other things, the following information for each object:

  • Orbital elements (semi-major axis, eccentricity, inclination, etc.)
  • Absolute magnitude and slope parameter
  • Number of observations and data-arc span
  • Epoch of the orbital elements
  • Orbit condition code

Planetoid-DB is a free and open-source software that allows users to visualize and analyze the orbital data of minor planets and comets in a user-friendly manner.

It is designed for astronomers, researchers, and enthusiasts who are interested in studying the dynamics of small celestial bodies in our solar system.

With Planetoid-DB, users can easily access and interpret the vast amount of data contained in the MPCORB.DAT file, enabling them to conduct research, track objects, and gain insights into the behavior of these fascinating celestial bodies.

Whether you are a professional astronomer or a hobbyist, Planetoid-DB provides the tools and resources needed to explore the wonders of our solar system and deepen your understanding of the small bodies that inhabit it.

For more information and to download the latest version of Planetoid-DB, please visit our official website or check out our GitHub repository.
